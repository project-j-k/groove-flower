# GROOVE FLOWER 🌻🕶️

音に反応して踊る花のブラウザトイです。80年代に流行した「音で踊る花のおもちゃ」への非公式オマージュとして、個人が趣味で制作したものです。

👉 **Demo:** https://<your-github-username>.github.io/groove-flower/

## 遊び方

- **AUTO** — ブラウザ内(Web Audio API)で合成した116 BPMのビートに合わせて自動で踊ります
- **MIC** — マイク入力の音量に反応して踊ります(手拍子や音楽をどうぞ)
- **STOP** — ひとやすみ

## 特徴

- 依存ライブラリゼロの単一HTMLファイル(画像・フォント・外部CDN不使用)
- 花はすべてSVGで描画(オリジナルの描き起こし)
- ビート(キック/スネア/ハイハット/ベース)はWeb Audio APIでリアルタイム合成
- 実物のトイのような「音のパルスでポーズがカクッと切り替わる」動きを再現

## プライバシー

MICモードのマイク音声は、お使いのブラウザ内で音量解析にのみ使用されます。
**録音・保存・外部送信は一切行いません。** 通信を行うコードは含まれていません。
